# PowerAnalysisESM

## Shiny app to perform power analysis to select the number of individuals in ESM studies

Download and run the shiny app:

```
library(htmltools)
library(shiny)
library(DT)
library(nlme)
library(ggplot2)
library(gridExtra)
library(data.table)
library(plyr)
library(dplyr)
library(formattable)
library(tidyr)
library(MASS)
library(plyr)
library(parallel)
library(shinyjs)

runGitHub("PowerAnalysisESM", "rstudio")
```
